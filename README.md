# Alura Med - API

Backend da aplicação para cadastro de pacientes.

## Instalação

Instalação das dependências.

```bash
npm install
```

Inicialização do DynamoDB para rodar localmente.

```bash
sls dynamodb install
```

## Uso

Rodando a aplicação localmente

```bash
sls offline start
```
